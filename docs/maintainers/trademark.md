# Thala Trademark Policy

The name **Thala** and associated logos, if any, are used to identify this project and its official releases.

## What You May Do

- Refer to this project by name in documentation, blog posts, and discussions.
- Say that your project "uses Thala" or "is built on Thala."
- Use the name in academic work, conference talks, or educational materials.

## What Requires Permission

- Distributing a fork under the name "Thala" in a way that could imply it is the official project.
- Using the Thala name or logo in product branding, paid services, or commercial offerings in a way that implies official endorsement.
- Creating a domain, social media account, or publication that could be mistaken for the official Thala project.

## Forking

Forks are welcome under the MIT/Apache-2.0 licenses. Forks should use a name that is clearly distinct from "Thala" to avoid confusion with the upstream project. Attribution to the original project is appreciated.

## Questions

If you are unsure whether a specific use is permitted, open an issue at
https://github.com/oswfrans/thala/issues and describe the intended use.
